Splunk SIEM Home Lab

Built a complete SIEM pipeline from scratch on a Windows host: provisioned an Ubuntu Server 22.04 VM in VirtualBox, deployed Splunk Enterprise 10.4.0, and configured the Universal Forwarder to ship Windows Security, System, and Application Event Logs into the SIEM in real time. Currently ingesting and analyzing 5,000+ live Windows events. Built a 3-panel security monitoring dashboard (login activity over time, failed logins by account, top event codes) and configured a scheduled SPL alert rule that detects brute-force login patterns (3+ failed attempts within a 15-minute window). Resolved real infrastructure challenges along the way, including VM network mode troubleshooting (NAT vs. Bridged Adapter), static IP configuration via Netplan, SSH-based remote administration of a headless Linux server, and Splunk credential recovery via user-seed.conf.
