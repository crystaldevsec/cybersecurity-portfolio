# Cybersecurity Portfolio — Crystal Martinez

Personal portfolio and landing page for Crystal Martinez, a cybersecurity professional with a background in IT support, GRC, and security awareness. The site showcases hands-on projects, technical skills, and certifications, and serves as a central hub for recruiters and collaborators to learn more and get in touch.

## Tech stack

Built with plain HTML, CSS, and JavaScript — no frameworks or build tools.

## Structure

This is a single-page site with the following sections:

- **About** — background, education, and focus areas
- **Projects** — hands-on cybersecurity and development projects
- **Skills** — tools, domains, and certifications
- **Contact** — social links and ways to connect
